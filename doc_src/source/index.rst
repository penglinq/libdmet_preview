.. libDMET

Home
****

.. image:: ../logo/pyscf-logo.png
   :width: 400

The library of Density Matrix Embedding Theory (libDMET) is an open-source
platform to develop and execute density matrix embedding for both periodic 
and molecular systems. 
In addition to its core libraries,  libDMET work within the PySCF ecosystem 
:ref:`installing_extproj`.

.. For installation instructions, see the :ref:`install` page.

.. For a guide to performing a variety of calculations with libDMET, see the :ref:`quickstart` guide.
